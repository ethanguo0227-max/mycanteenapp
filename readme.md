# ☀️ 暖阳食光 · 校园午餐聊天点评系统

一个温暖风格的校园午餐点评 + 聊天互动 Web 应用。用户可以发布午餐点评、上传照片、打分、进行地图导航，并在每条点评下实时聊天对话。

## ✨ 功能特性

- **用户登录** - 用户名登录，身份持久化保存
- **发布点评** - 支持文字、图片上传、综合评分（口味/温度/外观）
- **地图导航** - 集成高德地图链接，一键导航到餐厅
- **实时聊天** - 每条点评下都有独立聊天区，可互动交流
- **点赞功能** - 为喜欢的点评点赞
- **举报机制** - 举报不良内容，防重复举报
- **数据持久化** - 所有数据保存在浏览器本地存储中

## 🖼️ 界面预览

- 暖黄色调设计，仿校园午餐氛围
- 响应式布局，手机/电脑均适配
- 带头像的用户评论气泡

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| HTML5 | 页面结构 |
| CSS3 | 暖色风格、弹性布局、动画效果 |
| JavaScript (ES6) | 核心交互逻辑 |
| LocalStorage | 数据持久化存储 |
